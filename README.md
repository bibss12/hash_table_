# Hash Table Login System

## Deskripsi Project
Project ini merupakan aplikasi web sederhana yang mengimplementasikan struktur data Hash Table menggunakan JavaScript tanpa library eksternal.

Fitur:
- Menyimpan username dan password
- Mencari data user
- Menghapus data
- Menampilkan seluruh data user

Struktur data diimplementasikan secara manual menggunakan custom class.


## Cara Install Dependencies
Project ini tidak menggunakan dependencies atau library tambahan.


## Cara Menjalankan Project
1. Download atau clone repository ini
2. Buka file `index.html`
3. Jalankan menggunakan browser


## Teknologi
- HTML
- CSS
- JavaScript
